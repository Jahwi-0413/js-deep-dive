# js-deep-dive
모던 자바스크립트 deep dive 스터디

## 진행 규칙

1. 매주 각자 공부할 분량을 정하고 각자 공부한 내용을 정리한다.
2. 매주 일요일 저녁 8시에 리뷰한다.
3. 리뷰 할때는 공부하면서 헷갈렸던거나 같이 얘기해보고 싶은 것 등에 대해 얘기한다.
4. 내용 정리는 각자 이름의 폴더에서 진행한다.
